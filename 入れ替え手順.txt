おまもりみくじ Version 4.2　一式入れ替え手順

このZIPの中身が、GitHubへ入れる完成版一式です。

【入れ替え方法】
1. ZIPを展開します。
2. GitHub Desktopで使っている omamori-mikuji フォルダーを開きます。
3. 現在入っているファイルを、いったん別フォルダーへバックアップします。
4. リポジトリ直下にある既存ファイルを削除します。
   ※ .git フォルダーは絶対に削除しないでください。
5. このZIP内の以下8ファイルを、リポジトリ直下へコピーします。
   - index.html
   - apple-touch-icon.png
   - charomaru-miko.png
   - kamiyoshi.png
   - ureshiyoshi.png
   - en_yoshi.png
   - fukuyoshi.png
   - yuuyoshi.png
6. GitHub Desktopで変更を確認します。
7. Summaryに次を入力します。
   Version4.2 一式入れ替え・動作修正
8. Commit to main
9. Push origin

https://charomaru.github.io/omamori-mikuji/
